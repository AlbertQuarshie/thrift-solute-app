# Thrift-Solute

This project is the creation of a modern, user-friendly e-commerce platform designed to offer affordable and high-quality products for online shoppers. It is part of a web development project focused on building responsive and interactive online stores.

Thrift-Solute aims to simplify shopping by combining a clean interface with seamless browsing, cart management, and checkout processes.

## Contributor  
Albert Junior Quarshie  

### A Project Brief

Thrift-Solute is a web-based thrift store platform that allows users to browse products, add items to a cart, and place orders efficiently. The site features a responsive design, smooth navigation, and essential e-commerce functionalities to deliver a straightforward shopping experience.


### B. Core Features

#### 1. Product Browsing  
Users can browse available products categorized neatly with easy access from the home page and product listing pages.

#### 2. Shopping Cart  
Users can add or remove products to a cart with a dynamic total price update, enabling simple management before checkout.

#### 3. Checkout Process  
A straightforward checkout form allows users to review their order and place it securely.

#### 4. User Authentication  
Login and registration forms allow users to manage their shopping profiles and track orders.

#### 5. Responsive UI Design  
Built with Tailwind CSS, the platform is optimized for desktop and mobile screens for an accessible, clean look.



### C. Technologies Used
- React.js - Frontend framework for building UI  
- Tailwind CSS - Styling 
- JavaScript (ES6) - Logic and interactivity  
- React Router - Page routing and navigation  
- Local Storage - Cart data persistence  
- Vite (optional) - Development build tool  



### D. Usage Instruction
1. Clone the repository:  
git clone git@github.com:AlbertQuarshie/thrift-solute.git

2. Navigate to the project directory:
cd thrift-solute

3. Install dependencies:
npm install

4. Start the development server:
npm run dev

### E. Business Rationale

Thrift-Solute is designed to make thrift shopping accessible online, supporting sustainable shopping habits by offering curated products at affordable prices. The platform models how a simple but elegant e-commerce site can enhance user experience with minimal but effective features, catering to budget-conscious shoppers.
